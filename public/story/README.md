# Story chapter plates

Full-bleed stills for cold-start cinematic chapters (`HumanEconomyLanding` + `StoryChapterArt`).

| File | Chapter | Beat |
|------|---------|------|
| `ch-opening.webp` | Awareness | Attention is yours — notice before you scroll |
| `ch-problem.webp` | The Trap | Hours / distraction ≠ learning |
| `ch-awakening.webp` | Awakening | Human Passport from zero |
| `ch-spark.webp` | First Spark | Learn once with eyes open |
| `ch-evolution.webp` | Spread | Awareness compounds when shared |

Dashboard / deck panels use `MoodArt` **signal** plate (perspective lattice + sharp HUD glyphs) — not soft liquid photos.

Cold-start chapters may still fade a still under the lattice (`plate="photo"`). Prefer cyan / amber / rose / emerald accents.
